updaterエージェントとして、最新パッチに基づきチャンピオンガイドを更新してください。

## 手順

1. `current-patch.txt` を読んで現在の既知パッチバージョンを確認する
2. `patches/` ディレクトリを確認し、既知パッチより新しいファイルを特定する（例: patches/26.8.md）
3. そのパッチノートファイルを読んで、変更されたチャンピオンを特定する
4. 変更があったチャンピオンの `champions/[name]/guide.md` を読む
5. パッチノートの変更内容を反映してガイドを更新する（`POLICY.md` のフォーマット準拠）
   - パッチバージョン表記を更新する
   - バフ・ナーフ・変更内容を反映する
   - メタから外れた場合は `> ⚠️ パッチ XX.X 時点でメタ外。参考情報として残しています。` を追記する
6. 以下のJSON形式のみを標準出力に出力する（他の説明文は一切出力しない）：

```json
[
  {"op": "write", "path": "champions/[name]/guide.md", "content": "（更新後のガイド全文）"},
  ...
]
```

変更対象のチャンピオンが存在しない場合（ガイド未作成）はスキップする。
WebSearchは使用しない。パッチノートファイルの内容のみを参照する。
