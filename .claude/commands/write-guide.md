## 手順

1. `TODO.md` を読み、`- [ ]` の先頭チャンピオン（一番上の未完了）を特定する
2. そのチャンピオンについて WebSearch で最新情報を取得する（パッチ26.6対応、Mobalytics等を参照）
3. `POLICY.md` のフォーマットに従ってガイドを作成する
   - 「一言まとめ」「スキルオーダー＋理由」「主要パワースパイク」「ゲームプラン3フェーズ」「対策」
   - 簡潔に。1セクション3〜5行以内を目安
   - Mobalytics等を参考にしつつ丸パクリしない（要約・再構成する）
4. `champions/[champion-name]/guide.md` に保存する（champion-nameは英語小文字）
5. `TODO.md` の該当行を `- [ ]` から `- [x]` に更新する
6. 完了報告: 生成したファイルパスとチャンピオン名を出力する
