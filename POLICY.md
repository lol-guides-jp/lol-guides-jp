# コンテンツ方針

## コンセプト

「op.gg でビルドは見られる。でも"何をすべきか"がわからない」層向けの日本語クイックリファレンス。

Mobalytics 等の英語情報を参照しつつ、日本語で簡潔にまとめる（丸パクリ不可・要約・再構成）。

## ターゲット読者

- ランク帯: シルバー〜プラチナ
- JP1サーバープレイヤー
- 英語が読みにくい、または素早く要点を確認したい人

## フォーマット（1チャンピオン = 1ファイル）

```markdown
# [チャンピオン名]（ロール） パッチ XX.X

## 一言まとめ
チャンピオンの立ち位置を1〜2文で。「序盤強いアサシン」「後半化けるスケーラー」等。

## スキルオーダー
推奨: R > X > X > X
→ 理由を1行で。

## 主要パワースパイク
- Lv X: ○○習得で△△が可能になる
- Lv X: ○○マックスで△△が強くなる

## ゲームプラン（3フェーズ）
**序盤（〜15分）**: 何をすべきか1〜2文
**中盤（15〜25分）**: 何をすべきか1〜2文
**終盤（25分〜）**: 何をすべきか1〜2文

## 対策（このキャラを相手にするとき）
- ○○に注意: 理由1行
- ○○を活かす: 理由1行
```

## 更新ルール

- 対応パッチを明記する
- 大型バフ/ナーフがあったチャンピオンは次のセッションで優先更新
- 追加順: JPサーバー人気上位から

## 参照元

- Mobalytics（ゲームプラン・パワースパイク）
- op.gg / Lolalytics（統計・ティア）
- 公式パッチノート
